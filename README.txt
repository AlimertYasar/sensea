# Sensea

Sensea is an emotion-aware personal style intelligence system designed around how you feel.

It combines human-centered AI, emotion understanding, and visual style guidance to help users dress in a way that aligns with their inner state.

## Features
- Emotion-aware outfit recommendations
- Human-centered AI (server-side)
- Style and persona-based visuals
- Minimal, modern Android UI (Jetpack Compose)

## Tech Stack
- Android (Kotlin, Jetpack Compose)
- FastAPI backend
- Transformer-based emotion model (server-side)
